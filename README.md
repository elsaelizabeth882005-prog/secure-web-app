# Secure Web App Deployment

## Project Overview
This project demonstrates deployment of a secure web application on AWS using Docker, Nginx, and GitHub Actions.

## Application Theme
Student Capstone Project

## Technologies Used
- Python Flask
- Docker
- Nginx
- AWS EC2
- GitHub
- GitHub Actions (CI/CD)

## Project Structure
```text
secure-web-app/
├── app.py
├── requirements.txt
├── Dockerfile
├── README.md
└── .github/
    └── workflows/
        └── ci-cd.yml
```

## Features
- Custom student-themed Flask web application
- Docker containerization
- AWS EC2 cloud deployment
- Nginx reverse proxy
- CI/CD automation using GitHub Actions
- Monitoring and logging support

## Deployment Steps
1. Developed Flask application (`app.py`)
2. Added dependencies in `requirements.txt`
3. Created Dockerfile and built image
4. Ran Docker container on port 5000
5. Configured Nginx reverse proxy
6. Deployed application on AWS EC2
7. Implemented CI/CD pipeline with GitHub Actions
